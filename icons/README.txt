ICON PLACEHOLDER NOTICE
=======================

This extension requires icon files in PNG format:
- icon16.png (16x16)
- icon48.png (48x48)  
- icon128.png (128x128)

For testing, you can:
1. Create simple colored squares in any image editor
2. Or download Jupiter's actual logo and resize to these dimensions
3. Or use online tools like https://realfavicongenerator.net/

The extension will work without custom icons (Chrome will use default),
but proper icons improve the user experience.
